# India AQI Data Analysis (2015-2020)

## Overview
Analyzed India's Air Quality Index (AQI) dataset using MySQL 
to identify pollution trends, seasonal patterns, and city-wise 
comparisons across 26 Indian cities over 5 years.

## Tools Used
- MySQL 8.0
- Power BI
- Dataset: Kaggle (Air Quality Data in India)

## Key Insights
- Ahmedabad recorded the highest peak AQI among all cities
- Overall AQI improved consistently from 2015 to 2020
- 35% of all readings fall in the Moderate AQI category
- Winter months show significantly higher pollution levels
- Top 3 most polluted cities: Ahmedabad, Ahmedabad, Delhi

## SQL Queries Performed
- Average AQI by city
- Most polluted cities ranking
- Year-on-year AQI trend
- Pollutant contribution analysis
- Best air quality cities
- Monthly pollution patterns
- AQI category distribution
- City-wise yearly comparison
